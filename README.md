# Эхо (Echo)
**Текстовая мистическая игра о проклятом каньоне, демонах и цене освобождения.**
## Сюжет
Каньон проклят. Три демона заперты внутри. Ваша задача — найти правильное слово, чтобы освободить их... но готовы ли вы к последствиям?
## Управление
Запустите игру и вводите команды в консоль.
## Концовки
В игре три концовки:
*   Хорошая
*   Средняя
*   Плохая
## Как играть
1.  **Самый простой способ:** Скачайте файл `эхо.exe` и запустите его.
2.  **Запуск из исходного кода:** Установите Python и библиотеку `pygame` (`pip install pygame`), затем выполните команду `python эхо.py`. Потом надо положить mp3 файлы в одну папку с эхо.py.
